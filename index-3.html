🔥 Firebase Setup Required

This app uses Firebase Realtime Database so both you and your husband see the same data instantly. Setup is free and takes about 5 minutes.

1
Create a free Firebase project
Go to console.firebase.google.com → "Add project" → give it a name → Continue (disable Analytics is fine) → Create project.
2
Enable Realtime Database
In your project → left sidebar → Build → Realtime Database → "Create database" → choose a region → Start in test mode (you can secure it later) → Enable.
3
Get your config & paste it in the code
In Firebase console → ⚙️ Project settings → "Your apps" → click the Web icon (</>) → register app → copy the firebaseConfig object → open index.html in any text editor → replace the firebaseConfig block near the top (labeled YOUR FIREBASE CONFIG GOES HERE).
4
Host it so both of you can open the same URL
Run npx serve . on your computer, or drag the file to Netlify Drop, or deploy to GitHub Pages. Both you and your husband open that URL — data syncs in real time! ✨
